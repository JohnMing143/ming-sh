# Application definitions

`PROJECT_APPS_REPO` is intentionally empty. Until a personal application
repository is configured, the application menus retain the upstream
`kejilion/apps` repository through the centralized `UPSTREAM_APPS_REPO_URL`
setting in `config/project.conf`.

Application definitions are executable shell configuration. Review changes
before pulling or running them, and do not add telemetry or hidden update
services.
